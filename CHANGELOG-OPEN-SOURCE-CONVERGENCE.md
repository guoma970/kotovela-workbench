# 开源版页面收敛变更说明

**执行原则**：不做重构，只做轻量收敛优化。信息更直观、默认态更干净、适合截图展示。

---

## Dashboard（中枢总览）

| 收敛项 | 变化前 | 变化后 |
|--------|--------|--------|
| 页面说明文字 | 长段落含数据源说明 | 精简为 1 行 |
| Office Board 标题 | "言町科技实例工位图" + 详细说明 | "实例工位图" + 一行元信息 |
| Office Board 控制栏 | 含"返回上一步""手动刷新"等 5 个元素 | 只保留筛选按钮 + 倒计时 |
| 工位卡片元信息 | 含"工位示例"标签、角色、更新时间、状态说明、链接行 | 只保留：名称、状态、当前任务、关联项目 |
| 英雄区（主项目） | 分两行显示"当前重点"+"下一步"+跨页链接行 | 合并为一行，下一步附在后 |
| 英雄区元信息 | 4 项含"下一步" | 3 项，Blocker 标红 |
| 实例统计网格 | 4 个 StatCard 卡片（重复 Office Board 数据） | 已移除 |
| Blocker 标题 | "Blocker 区 / N 个阻塞" | "Blocker / N" |
| Blocker 项说明 | 含更新时间、任务单标识、3 个跨页链接 | 只保留：标题、优先级、项目+实例徽章、1 个链接 |
| 待拍板 / 最近更新面板 | 每项含多个链接行 | 只保留标题、对象徽章、时间和类型标签 |
| 跨页映射区 | 整块区域含说明段落 | 已移除（降级到下半区或折叠） |

---

## Agents（实例状态）

| 收敛项 | 变化前 | 变化后 |
|--------|--------|--------|
| 页面说明 | 长段落含数据源说明 | 精简为"一眼看出谁卡住/谁在干/谁空着" |
| 页面指标 | 含"平均挂载任务"等 5 个 | 精简为 4 个（实例/活跃/阻塞/待命） |
| 卡片元信息条 | 含角色、更新时间两行 | 已移除 |
| 关联关系堆叠 | 含项目、房间、任务全部展开 | 只保留：项目徽章 + 房间数/任务数统计 |
| 卡片底部 | 2 个跨页链接"查看相关项·Tasks/Rooms" | 1 个链接"查看关联任务" |
| 每张卡字段顺序 | 名称→状态→角色/时间→任务→项目/房间/任务 | 名称→状态→当前任务→项目→更新时间/统计 |

---

## Tasks（任务队列）

| 收敛项 | 变化前 | 变化后 |
|--------|--------|--------|
| 页面说明 | 长段落含数据源说明 | 精简为"Blocker 优先 → Doing 推进 → Done 归档" |
| 面板指标 | 4 个指标 | 4 个指标（简化文案） |
| 任务卡底部 | 含 2 个跨页链接 + 负责人/更新时间行 | 只保留：优先级徽章、项目+实例徽章、负责人+时间 |
| 任务卡字段顺序 | 标题→优先级→任务单→项目→实例→房间→链接行→时间 | 标题→优先级→项目+实例→负责人+时间 |
| 列标题 | "Doing / N 条" | "Doing / N" |
| 空状态文案 | "当前没有任务。" | "无任务" |

---

## Projects（项目看板）

| 收敛项 | 变化前 | 变化后 |
|--------|--------|--------|
| 页面说明 | 长段落含数据源说明 | 精简为"每个项目含状态、阶段、阻塞数、成员和关联任务" |
| 项目卡元信息 | 4 项含"任务量" | 3 项，去掉"任务量" |
| 项目卡重点区 | 分"当前重点"+"下一步"两行 | 合并为一行，重点在上，下一步在→后 |
| 关联任务 | 全部展开 | 最多显示 3 条，多余显示"(共N)" |
| 关联实例/房间/任务标签 | "关联实例"/"承接群"/"关联任务" | "实例"/"房间"/"关联任务" |
| 跨页链接行 | 3 个链接 Chips | 1 个"关联任务"链接 |
| 进度条 | 底部含进度条 | 已移除 |

---

## Rooms（群与通道状态）

| 收敛项 | 变化前 | 变化后 |
|--------|--------|--------|
| 页面说明 | 长段落含数据源说明 | 精简为"每个房间展示：名称、状态、当前用途、最近动作类型" |
| 卡片元信息 | 含"通道类型"+"待处理项" | 含"状态"+"待处理" |
| 房间卡关联 | 含项目、实例、任务全部展开 | 只保留：项目 + 实例徽章 |
| 关联任务 | 全部展开 | 已移除 |
| 跨页链接行 | 3 个"查看相关项·XXX" | 1 个"关联任务"链接 |
| 最近动作 | 在 relation-stack 底部普通显示 | 在 info-block 中突出显示 |

## 统一样式

- 新增 `.badge-count`：面板标题右侧的紧凑计数徽章
- 新增 `.empty-compact`：精简空状态文案
- 新增 `.text-red`：红色文字（Blocker 计数）
- 新增 `.error-tag`：错误提示样式
