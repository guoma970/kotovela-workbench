# Public example only

The real Kotovela Hub execution assets were moved to the private repository:

- Private repo: `https://github.com/guoma970/kotovela-hub`
- Public repo: example payload and forced dry-run wrapper only

## Files kept in public

- `payload.example.yaml`
- `run_payload_batch_sync.sh`

## Behavior

- any payload other than `payload.example.yaml` is rejected
- `real_run=true` is rejected
- embedded tokens / Authorization values are rejected
- the script only prints the example payload and exits as a dry-run preview
