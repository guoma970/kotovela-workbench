# Security Policy

## Public-safe boundary

`openclaw-kotovela` is a public-safe, mock-only showcase for the OpenClaw × Kotovela cockpit. The public demo is https://openclaw-kotovela.vercel.app.

Do not commit private runtime files, live workspace adapters, real service endpoints, customer data, private local paths, chat/user identifiers, or tokens. Keep all examples synthetic and replace any sensitive value with an obvious placeholder before opening a pull request.

## Reporting

If you find a secret, private identifier, or live payload in this public repository, do not reuse it. Open a GitHub security advisory or contact the repository owner so the value can be rotated and removed from history if needed.
